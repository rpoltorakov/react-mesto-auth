# Mesto-react-auth

### **React**-Проект "Место"

## Реализовано:

* Попап редактирования профиля
* Попап добавления карточки
* Попап редактирования аватара профиля
* Попап полноэкранной карточки
* Карточки и профиль загружаются с сервера
* Регистрация и аутентификация на сервере
